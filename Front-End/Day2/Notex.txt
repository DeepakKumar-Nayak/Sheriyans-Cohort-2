HTTP Request Response Cycle:
-----------------------------
HTTP: Hyper Text Transfer Protocol.
HTTP allow karta hai data ko bhejna and data ko mangana
for example jab aap message send karte ho and jab apko message 
milta hai whatsapp mein yeah sab kuch ho raha hao HTTP se 

. har browser HTTP Rules follow karta hai. 
HTTP Request Response Cycle mein 4 steps hai 
1. Request 
2. Processing
3. Response
4. Render

sabse pehle client kya karta hai request karta hai then server 
apke request ko read karta hai and jo bhi data apne manga hai usko 
bhej deta hai

for example mein ChatGPT se request kiya 2+2 kya hota hai so chatGPT 
ne usko process kiya and response kiya ki 2+2 hota hai 4 

jab bhi ham browser mein youtube.com likhte hain tab browser ek 
HTTP request bhejta hai server ko(youtube ke server ko) then kya hota 
hai then wahan pe request accept hoti hai and then processing hoti hai 

processing hoti hai in the sense ki youtube server dekhta hai ki ki tumne 
request bheji kya hai suppose tumne bheja hai ki mujhe kuch funny video de do 
toh woh uske databse se leke apko bhej dega 10 funny videos esko kehte hai 
response pehle processing hua data mil gaya and then bhej diya 

kiske pass bhej diya user ke pass jis mein hota hai HTML CSS Javascript,
images, photos etc etc 

aab 4th step follow hoha jo hai render eska matlab jo apko response mila hai 
usko show karna.
